## 优化内容概述

### 新增模块：鲁棒性设计 (Robustness)

为所有 8 个角色新增了统一的鲁棒性设计模块，包含：

1. **歧义处理机制** - 针对各角色常见歧义场景的处理方式和示例
2. **任务失败恢复机制** - 5种以上失败场景及对应恢复策略
3. **降级策略** - 最小/标准/完整三级降级输出方案
4. **角色特定增强**：
   - 产品经理/需求分析师：需求变更管理流程
   - 架构师：常见架构反模式警示
   - 项目经理：项目变更管理流程
   - 前端/后端工程师：性能指标量化表
   - 测试工程师：测试失败处理流程 + 覆盖率指标
   - DevOps工程师：灾难恢复计划 (DRP) + 运维指标

### Gate Check 量化增强

为所有角色的质量检查清单增加了量化指标，例如：
- 产品经理：MVP 功能数 ≤ 5，P0 功能 ≤ 30%
- 前端工程师：LCP ≤ 2.5s，any 类型数量 = 0
- 后端工程师：P99 ≤ 200ms，NPE 风险 = 0
- 测试工程师：P0 通过率 = 100%，P1 ≥ 95%
- DevOps工程师：MTTR ≤ 30min，可用性 ≥ 99.9%
